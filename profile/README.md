# PhotoDewey

**A free photo library manager for Windows.**

Organize your photos with keyword tags, make non-destructive edits, browse your collection on a map, and keep your metadata in sync — all stored locally on your own machine.

→ **[photodewey.eu](https://photodewey.eu)** — full overview, screenshots, and release notes

The installer is per-user — no administrator prompt. The app keeps itself up to date, but you can use *Help → Check for Updates…*

> **On install:** Windows shows a SmartScreen warning because the beta installer isn't yet code-signed. Click **More info → Run anyway**. The installer only ever writes to your user profile.

## What PhotoDewey does

- **Organize** — Import photos into a structured folder layout with automatic file naming based on date, camera, or sequence number.
- **Tag** — Build a hierarchical tag tree (`places/europe/italy`) and assign tags to one photo or hundreds at once. Tags are written back into the photo files so other software can read them.
- **Edit non-destructively** — Apply tone adjustments, exposure corrections, filters, rotations, and flips. The original file is never touched; edits are stored separately and applied on the fly.
- **Map** — Browse your collection geographically. Photos with GPS coordinates appear as pins on an interactive map; you can assign a location to photos that don't have one.
- **Extend** — Install community-built plugins to add new editing tools. Drop a plugin into a folder and PhotoDewey picks it up automatically.

## Getting help

**Something went wrong?**
Use *Help → Send Feedback* inside the app — it attaches your version, OS, and an anonymous summary automatically. Or open a [bug report](../../../../PhotoDewey-for-Windows/issues/new/choose) here on GitHub if you'd like a public conversation other users can follow.

**Have a suggestion?**
Check the [Issues tab](../../../../PhotoDewey-for-Windows/issues/) first to see if it's already there — a 👍 reaction on an existing request carries more weight than a duplicate. If it isn't listed, open a [feature request](../../../../PhotoDewey-for-Windows/issues/new/choose).

**Just a question?**
The in-app User Manual (*Help → User Manual*) covers importing, tagging, editing, exporting, and more. For anything else, open a [workflow question](../../../../PhotoDewey-for-Windows/issues/new/choose).

---

## Privacy

PhotoDewey stores everything locally — your library, thumbnails, settings, and catalog. Nothing is uploaded automatically.

*Help → Send Feedback* sends only what you type, your app version, OS info, and a count of photos and tags in your library (no filenames, no paths, no GPS data, no images). You can opt out entirely on the Beta Welcome screen.
